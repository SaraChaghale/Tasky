# Tasky - App de Listas de Tareas

**Tasky** es una aplicación nativa para Android diseñada para la gestión eficiente y personalizada de tareas diarias. El objetivo es ofrecer una herramienta intuitiva que ayude a los usuarios a organizar su día a día de forma sencilla.

##  Características principales
* **Listas personalizadas:** Crea y organiza tareas según tus necesidades.
* **Interfaz intuitiva:** Diseño moderno centrado en la experiencia de usuario (UX).
* **Gestión dinámica:** Posibilidad de añadir y completar tareas en tiempo real.
* **Persistencia de datos:** (Opcional: añade si usas Room o SQLite) Tus tareas se guardan de forma local.

##  Tecnologías utilizadas
* **Lenguaje:** Kotlin
* **Entorno de desarrollo:** Android Studio
* **Arquitectura:** (Ejemplo: MVVM / Clean Architecture)
* **Componentes:** Jetpack (ViewModel, LiveData), Room (si aplica).
